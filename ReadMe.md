# kiwibook par Estelle Corsetti et Simon vivier

## Résumé des fonctionnalités

### Le chat
Permet d'afficher les messages envoyer dans le chat et également d'en envoyer.

### Le mur
Le mur affiche les messages postés par les utilisateurs. 
Une méthode de chargement par requête AJAX permet d'afficher les messages petit à petit.

### Le profil
Il est possible de consulter et d'éditer son profil.

### Les amis
Il est possible de consulter la liste des amis, ici la liste des utilisateurs.
